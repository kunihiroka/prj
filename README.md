# Induction Motor Regeneration Instability

Analysis repository for induction-motor regeneration instability, including reports, plots, MATLAB scripts, and A3 presentation material.

## Contents

- `induction_motor_regen_instability/`: Main analysis package and generated report artifacts.
- `induction_motor_regen_instability/README.md`: Detailed project report and explanation.
- `induction_motor_regen_instability/induction_motor_regen_instability_report.pdf`: Rendered report.
- `induction_motor_regen_instability/*.png`: Supporting plots and block diagrams.
- `induction_motor_regen_instability/scripts/`: MATLAB scripts for frequency-response and open-loop analysis.
- `a3/`: A3 presentation deliverable.
- `tmp/`: Project-local temporary material.

## Notes

This repository already tracks the detailed report inside `induction_motor_regen_instability/README.md`. The root README is a short navigation entry point.
